A Package

This drawing was created in Procreate using a brush that I do not typically use for digital illustrations. I made this for a drawing assignment where we could draw anything, in the animating software of choice, just to get familiar with the tools and interface. I started this drawing with an isometric approach because I wanted to animate the tree growing, which was another assignment, but I decided that I wanted something else. I experimented a lot with this drawing, with the tree and the house influencing or limiting the composition. The window frame developed towards the end of the drawing after experimentation with trying to add something different to just a straight-on isometric view, which I have done a lot of.
